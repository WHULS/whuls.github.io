图片加载

在页面中加载图片有三种方式

  1. 使用 JavaScript 的 Image 对象
  2. 写在 <img> 标签中
  3. 在 css 中用 url() 引用

无论是哪种方式，利用 webpack 进行图片打包时，都需要用到 file-loader

在 webpack.config.js > module.exports > module 中引入

打包 <img> 要配合 html-loader 使用，打包 css 则需要 style-loader 和 css-loader
