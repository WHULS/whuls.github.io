Hexo修改标签大小写

发表于 2020-11-12 更新于 2021-04-09 分类于 Git
本文字数： 266 阅读时长 ≈ 1 分钟
  1. 问题描述
  2. 原因及解决

问题描述

在Hexo中修改标签的大小写后，之前提交到代码仓库的tags文件夹下的子文件夹并不会发生变化，但是在tags页面下的标签名称是会发生变化的，因此点击标签进入页面后，提示404。

原因及解决

https://vopaaz.github.io/2019/01/15/change-tag-case-in-Hexo/

原因是git对文件名的大小写不敏感，因此只需要修改git配置，再重新部署一次，就可以解决问题。

重新部署后建议对文章进行一些修改，否则hexo无法替换原来的文件
